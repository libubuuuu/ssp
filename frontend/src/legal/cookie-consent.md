## 1. 什么是 Cookie

Cookie 是当您访问 {{PRODUCT_DOMAIN}} 时,我们的服务器在您浏览器(或设备)上存储的一小段文本数据。Cookie 在您下次访问时被读取,用于识别您、维持登录态、记住您的偏好。

类似技术包括但不限于:**localStorage**、**sessionStorage**、**IndexedDB**、**Web Beacon**(本政策统称为"Cookie")。

---

## 2. 我们使用的 Cookie 类型

依据用途,我们将使用的 Cookie 分为四类:

### 2.1 严格必要 Cookie(默认开启,关闭后服务无法使用)

| Cookie 名称 / 用途 | 说明 | 有效期 |
|---|---|---|
| `access_token`(httpOnly) | 维持您的登录态 | 1 小时 |
| `refresh_token`(httpOnly) | 在 access_token 过期前刷新登录,避免频繁掉线 | 30 天 |
| Session 状态(localStorage)| 用户基本信息缓存(姓名、头像、积分),用于 UI 显示 | 直至登出 |
| CSRF 防护 Cookie(若有) | 防跨站请求伪造攻击 | 单次会话 |

> 这些 Cookie 用于**核心安全 / 登录功能**,关闭后您将无法使用本服务。

### 2.2 功能性 Cookie(默认开启,关闭不影响主要功能)

| Cookie 名称 / 用途 | 说明 | 有效期 |
|---|---|---|
| `lang`(localStorage) | 记录您的语言偏好(中文 / 英文) | 持久 |
| UI 偏好(侧边栏开/收等) | 优化您的界面体验 | 持久 |

### 2.3 分析 Cookie(可选,需您同意后开启)

| 服务 | 用途 | 隐私 |
|---|---|---|
| Sentry(若启用)| 错误监控,定位线上 bug | 仅记录异常堆栈与 trace_id,**不**记录您的输入内容 |
| 自建访问日志 | 排查故障,分析使用模式 | 服务器端日志,无浏览器侧 Cookie |

如您同意,我们将开启 Sentry 客户端 SDK 收集前端报错。**关闭后我们仍可通过服务器日志观察后端故障**,但前端报错信息会丢失。

### 2.4 营销 / 广告 Cookie

**我们目前不使用任何营销 / 广告 / 第三方追踪 Cookie。**

我们承诺:
- 不投放任何第三方广告 SDK
- 不与广告联盟共享您的浏览数据
- 不基于您在本服务的行为做跨站追踪

如未来政策变化(例如开放广告功能),我们将提前更新本政策并征得您的明确同意。

## 3. 第三方 Cookie

通过本服务,以下第三方可能在您的浏览器设置 Cookie:

### 3.1 Cloudflare(若启用 CDN)

- 用于 DDoS 防护、流量优化
- 设置 `__cf_bm` 等防机器人 Cookie
- 详见 [Cloudflare Cookie 政策](https://www.cloudflare.com/legal/cookie-policy/)

### 3.2 第三方支付通道(若启用)

- 仅在您主动发起充值时跳转支付页面
- 第三方设置的 Cookie 受其自身政策约束(微信支付 / 支付宝等)

我们**不能**控制第三方 Cookie 的具体行为。如您关注隐私,请同时查阅相关第三方的政策。

## 4. 您的选择

### 4.1 浏览器设置

您可通过浏览器设置控制或拒绝 Cookie:

- **Chrome**:`设置 → 隐私设置和安全性 → Cookie 及其他网站数据`
- **Firefox**:`设置 → 隐私与安全 → Cookie 和网站数据`
- **Safari**:`偏好设置 → 隐私 → Cookie 和网站数据`
- **Edge**:`设置 → Cookie 和网站权限`

> ⚠ 如您**拒绝所有 Cookie**,本服务的核心功能(登录、付费、生成历史)将无法使用。

### 4.2 我们的同意管理

您首次访问本站时,会看到 Cookie 同意横幅:

- **接受全部**:开启严格必要 + 功能 + 分析 Cookie
- **仅必要**:仅开启严格必要 Cookie,功能 / 分析关闭
- **管理偏好**:逐项开关

您可随时在 `个人中心 / 隐私偏好` 调整设置。变更将于下次访问生效。

### 4.3 删除已存 Cookie

清除 {{PRODUCT_DOMAIN}} 已存 Cookie 的方式:

- 在浏览器隐私设置中点击"清除特定网站数据"输入 `{{PRODUCT_DOMAIN}}`
- 在我们的"个人中心 / 安全 / 登出所有设备"统一失效服务端 token
