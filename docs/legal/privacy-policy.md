## 3. 我们如何使用您的个人信息

### 3.1 提供服务

- 验证您的身份并维持登录状态
- 接收您的提示词与上传文件,调用 AI 模型生成内容
- 记录您的生成历史,供您随时查看下载

- 分析使用日志,定位故障、优化性能
- 检测异常访问(同 IP 短时高频注册、暴力破解等),保护账号安全

### 3.3 履行法律义务

- 配合公安机关、网信办等监管部门的合法查询
- 遵守《网络安全法》《个人信息保护法》《数据安全法》要求保留必要日志

### 3.4 我们**不**会用您的数据做的事

- **不**会未经您单独同意将您的个人信息用于商业广告投放
- **不**会将您上传的图片、视频用于训练第三方 AI 模型
- **不**会出售您的个人信息

## 5. 个人信息的存储

### 5.1 存储地点

主要数据存储在中国境内服务器(目前部署于腾讯云轻量服务器,具体地域见 ICP 备案信息)。

跨境传输的字段(见 §4.1)在第三方服务商境外数据中心处理后即时删除或仅保留必要时长。

### 5.2 存储期限

| 数据类型 | 期限 |
|---|---|
| 账户基本信息 | 您的账户存续期间 |
| 生成内容历史 | 您的账户存续期间(您可随时手动删除单条记录) |
| 上传媒体文件(归档至 /uploads/)| 90 天后自动清理(`uploads_gc` cron) |
| 网络访问日志(IP / UA) | 6 个月(《网络安全法》最低要求 6 个月) |
| 注册 / 登录失败日志 | 24 小时(反爆破窗口) |
| 充值订单记录 | 5 年(税务 / 财务合规) |
| 已注销账户的记录 | 注销后立即匿名化或删除,审计日志保留 3 年 |

### 5.3 安全措施

我们采用以下措施保护您的个人信息:

- 密码采用业界标准 bcrypt 哈希存储,**永不**以明文形式持有
- JWT Token 与 Refresh Token 通过 httpOnly Cookie 传递,防 XSS 窃取
- 所有 API 通信走 HTTPS / TLS 1.2+
- 数据库文件采用文件系统级权限隔离(非 root 业务账户访问)
- 关键密钥(JWT_SECRET / FAL API Key)加密保管,定期轮换
- 服务器、内部账户启用二步验证(2FA)
- 部署 nginx / fail2ban 防 DDoS / 暴力破解
- 每日异地加密备份,定期演练数据恢复

尽管如此,**互联网环境无法保证 100% 安全**。如发生数据泄露事件,我们将依法及时通知您与监管机构。

## 6. 您的权利

依据《个人信息保护法》第 44-49 条,您对自己的个人信息享有以下权利:

### 6.1 访问权
登录账户后,您可在 `个人中心 / 历史记录` 查看自己的所有数据。

### 6.2 更正权
如发现信息有误,可通过 `个人中心 / 修改资料` 自行更正,或联系 {{CONTACT_EMAIL}}。

### 6.3 删除权
您有权要求删除自己的个人信息,我们将在收到请求后 **15 个工作日内** 完成处理。删除范围不包括法律要求保留的最低数据(如交易记录、网络日志)。

### 6.4 撤回同意权
您可随时撤回先前授予的同意。撤回后,我们将停止处理对应数据,但不影响撤回前的合法处理。

### 6.5 注销账户
登录后在 `个人中心 / 安全 / 注销账户`(如已上线该功能)或邮件至 {{CONTACT_EMAIL}} 申请注销。注销后您的账户将无法恢复,所有内容将被删除。

### 6.6 复制权 / 转移权
您可申请获得自己个人信息的副本(机器可读格式)。请通过 {{CONTACT_EMAIL}} 提交申请。

### 6.7 投诉权
若您认为我们的个人信息处理行为侵犯了您的合法权益,可向中央网信办举报中心(www.12377.cn)或所在地网信、公安部门投诉。
