# 法务文档(草案)

> ⚠ **重要**:本目录所有文件**仅供参考,使用前必须经过用户企业法务 / 律师审阅**。
>
> 本目录用于支持 ICP 备案 / 公安备案 / 个人信息保护法 PIPL / 生成式 AI 管理办法等合规要求。
> Claude(AI 助手)起草,基于公开法规 + SSP 项目实际业务模型,**非法律意见**。
