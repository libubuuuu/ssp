用户的时间比 token 算力宝贵。该决断时直接干,不要每轮列 3-4 个选项让用户挑。

**Why:** 2026-04-28 用户在第九轮明确说"冷静思考后在干活,你要记住我的时间比算力还要宝贵,所以你要做得好也要速度快"。前 8 轮我每轮在结尾列"剩下 X、Y、Z 选哪个",让用户重复挑选 = 浪费用户时间。

**How to apply:**
- 该选择时,自己**冷静想清楚**最高 ROI 的一件事就开干,不要列菜单让用户挑
- 选项菜单只在两种情况列:
  1. 真要动钱 / 破坏性操作 / 跨大版本(需要用户拍板)
  2. 用户明确问"还有什么可做"(信息查询)
- 收尾不要每轮都问"要继续吗,选什么",用一句话留口子即可
- 解释要紧凑,不要把决策思路全 dump 出来 — 用户看的是结果不是思考过程
- 速度跟对正确并重:别为快牺牲质量,别为质量啰嗦磨蹭
